# Breakout Game in x86 Assembly

A fully working Breakout-style game built in x86 Assembly using BIOS interrupts.

## Features
- 5 levels with increasing speed
- Power-ups
- High score tracking
- Sound effects via PC speaker
- ASCII-based colorful UI
- Paddle and ball physics

## Controls
- Left Arrow  → Move paddle left
- Right Arrow → Move paddle right
- ESC         → Exit

## Tech Stack
- x86 Assembly (NASM)
- BIOS interrupts
- Text-mode graphics (0xB800)
